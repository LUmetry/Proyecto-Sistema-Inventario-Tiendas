# Skill: Sistema de Inventario en Consola (C#)

## Descripción del Proyecto

Este proyecto es un sistema de inventario básico desarrollado en C# para ejecutarse en consola. Su propósito es educativo, por lo que no busca ser altamente robusto ni escalable, sino claro y fácil de entender.

El sistema permite:

* Agregar productos
* Mostrar lista de productos
* Actualizar cantidades en inventario
* Eliminar productos
* Realizar operaciones simples de control de stock

## Tecnologías Usadas

* Lenguaje: C#
* Tipo de aplicación: Consola (.NET)

## Estructura del Proyecto

El código se organiza de forma simple, pudiendo incluir:

* Clases para representar entidades (Producto, Inventario, etc.)
* Métodos dentro de Program.cs o clases auxiliares
* Uso de listas en memoria (List<T>) para almacenar los datos

No se utiliza base de datos ni arquitecturas complejas.

## Objetivo de la IA

La IA debe ayudar a:

* Escribir código claro, sencillo y funcional
* Explicar conceptos si es necesario
* Proponer soluciones simples y entendibles
* Evitar sobreingeniería

## Reglas y Convenciones

* Usar nombres en español (clases, variables, métodos)
* Mantener el código lo más simple posible
* Priorizar claridad sobre optimización
* Usar estructuras básicas (if, switch, loops, listas)
* Evitar patrones avanzados innecesarios

## Instrucciones para la IA

* Generar código acorde a un nivel educativo (principiante/intermedio)
* Explicar brevemente cuando algo no sea evidente
* No usar librerías externas innecesarias
* Mantener todo funcional dentro de consola
* Respetar la lógica simple del sistema

## Qué evitar

* Uso de bases de datos
* Arquitecturas complejas (MVC, Clean Architecture, etc.)
* Código demasiado avanzado o difícil de entender
* Optimización innecesaria
* Uso excesivo de clases o abstracciones

## Contexto adicional

Este proyecto está diseñado para practicar lógica de programación y estructura básica en C#. Todo el manejo de datos se realiza en memoria y se pierde al cerrar el programa.
